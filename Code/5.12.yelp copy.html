Feature Engineering

Reviewer Activity:

  • Number of reviews written (from user_df)
  • Frequency of review posting (calculate from review_customer_df)

Review Quality:

  • Length of reviews (calculate from review_customer_df)
  • Use of specific details (e.g., product attributes; need to run a nn model)
  • Presence of extreme language (polarized sentiment; need to run a nn model)

Engagement Metrics:

  • Number of helpful votes or likes (from user_df and review_df)
  • Number of reported/spammed reviews (base data not provided)

Reviewer Consistency:

  • Variance in star ratings given by the reviewer (from review_customer_df)
  • Alignment of their ratings with overall product sentiment (wait till you run the nn model)

Historical Behavior:

  • Account age and tenure on the platform (calculate from user_df)
  • Whether reviews have been flagged or removed for violating guidelines (base data not provided)
